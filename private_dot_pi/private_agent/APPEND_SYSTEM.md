# USING OPENSPACE SKILLS

You also have access to the OpenSpace MCP server via `mcporter`.

## OpenSpace tools

OpenSpace provides the following tools: `execute_task`, `search_skills`, `fix_skill`, and `upload_skill`. Use `npx mcporter list openspace` to see all of them in detail.

## How to use OpenSpace

<IMPORTANT>

When attempting to complete a task, first determine if Superpowers skills or other registered skills can be used to complete the task. If not, use OpenSpace MCP server as fallback.

</IMPORTANT>
